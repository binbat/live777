# OBS Studio

可以用 OBS Studio 来推流

::: tip
1. OBS Studio 版本大于 [**30 or higher**](https://obsproject.com/forum/threads/obs-studio-30-beta.168984/)
2. OBS WHIP 目前只支持 **H264** 视频编码和 **Opus** 音频编码
3. [关于 OBS AV1 编码无法播放的问题](https://github.com/binbat/live777/issues/169)
:::
